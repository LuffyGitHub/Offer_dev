# 云开发数据库部署指南

## 1. 前置条件

1. 小程序 AppID 必须是**企业或个人主体**（非测试号）
2. 在[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)中打开项目
3. 顶部工具栏 → **云开发** → 开通并创建环境

## 2. 配置云环境 ID

打开 `utils/cloud.js`，将 `ENV` 替换为你的云环境 ID：

```js
const ENV = 'your-env-id'  // → 改为你的环境ID，如 'dev-xxxxx'
```

## 3. 创建数据库集合

在微信开发者工具 → 云开发 → 数据库 中，创建以下 7 个集合：

| 集合名 | 说明 | 权限 |
| :--- | :--- | :--- |
| `users` | 用户信息（昵称、头像、积分、段位） | 所有用户可读，仅创建者可写 |
| `user_profiles` | 用户画像（年级、专业、目标） | 所有用户可读，仅创建者可写 |
| `level_progress` | 关卡进度记录 | 所有用户可读，仅创建者可写 |
| `wrong_questions` | 错题本 | 仅创建者可读写 |
| `sign_in` | 签到记录 | 仅创建者可读写 |
| `badges` | 已解锁勋章 | 仅创建者可读写 |
| `settings` | 用户设置 | 仅创建者可读写 |

## 4. 部署云函数

在开发者工具中：
1. 左侧栏右键 `cloudfunctions/` → **更多** → **云函数增量上传**
2. 或者逐个右键每个云函数文件夹 → **上传并部署**
3. 等待部署完成（每个函数约 10-30 秒）

需要部署的云函数：
- `user` — 用户登录、获取数据、保存画像和设置
- `game` — 闯关进度、排行榜
- `score` — 积分变动、签到、勋章
- `wrongbook` — 错题本 CRUD

## 5. 验证

部署完成后，重新编译小程序，首次登录时数据会自动写入云数据库。
